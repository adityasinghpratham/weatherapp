This project is a complete combination of HTML, CSS and JavaScript. The idea of this project is quite simple it consists of a search bar where you have to enter the name of a place in order to the weather details of that place instantly. For weather details I used https://openweathermap.org/api API(Application Programming Interface). It helps a lot to get the weather details such as temperature, type of weather (sunny, smoky, foggy, rainy). Weather API provides a key in order to access weather details. SO, as soon as the user enters the name of the city it checks whether the entered city is valid or not if valid then details are generated. The background image changes according to the type of weather (sunny, smoky, foggy, rainy) which helps user to visualize the weather details.
